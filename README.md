# lancheria Vue JS

just a small project for vue feeding

## Setup Project Step-by-Step:


1. ##### Clone the repo:

```
git clone https://github.com/Lukas-xrz/lancheriaVueJS
```

```
cd lancheriaVueJS
```


2. ##### Install npm:

```
npm install 
```


3. ##### Install vue via npm:

```
npm install -g @vue/cli
```

4. ##### Run the project using: 
```
npm run serve
```
